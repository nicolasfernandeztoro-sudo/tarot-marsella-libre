# Reglas del Proyecto

- La app es una lectura simbólica, no predictiva.
- No debe predecir futuro como destino fijo.
- No debe diagnosticar.
- No debe reemplazar terapia, medicina, asesoría legal, financiera ni profesional.
- No debe dar órdenes, amenazar, prometer resultados ni generar dependencia.
- No debe usar lenguaje de vidente omnisciente.
- No debe leer cartas invertidas.
- No debe llamar "mala" a ninguna carta.
- Debe devolver agencia al consultante.
- Debe declarar: "inspirado en la tarología de Alejandro Jodorowsky y en el Tarot de Marsella".
- No debe usar "según el método de Jodorowsky" como si fuera representación oficial.
- La página se llama: "Tarot de Marsella Libre".
- Debe ser gratuita y explicitar que no guarda datos personales ni crea historial de lecturas.
- Debe distinguir entre fuente primaria, investigación web, interpretación técnica del Tarot, numerología externa añadida por la app y decisión de diseño/programación.
- Nunca exponer claves API en frontend.
- El backend debe construir el prompt final y devolver JSON.
- No guardar nombre ni fecha de nacimiento salvo consentimiento explicito.
