# Tarot de Marsella Libre

Aplicación web gratuita de lectura simbólica inspirada en la tarología de Alejandro Jodorowsky y en el Tarot de Marsella.

> Lectura simbólica inspirada en la tarología de Alejandro Jodorowsky y en el Tarot de Marsella. No predice el futuro, no diagnostica y no sustituye ayuda profesional.

El espacio no cobra, no guarda datos personales y no crea historial de lecturas.

## Estado

Proyecto en fase de preparación y diagnóstico. La implementación completa de la app, la baraja estructurada y la conexión con IA quedan para fases posteriores.

## Estructura

- `index.html`: entrada futura de la app.
- `src/`: código frontend modular.
- `worker/`: backend futuro para construir prompts y devolver JSON.
- `docs/`: investigación, criterios éticos y especificación.
- `original/`: copia del HTML original de referencia.
- `cartas/`: imagenes disponibles de cartas.
